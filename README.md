# Caça ao tesouro - Kotlin - Jetpack Compose

Este projeto é um jogo de Caça ao Tesouro desenvolvido em Android nativo com Jetpack Compose.

## Funcionalidades
- Navegação entre múltiplas telas de pistas.
- Validação de respostas para avançar.
- Cronômetro para medir o tempo total de jogo.
- Animações de transição entre telas.
